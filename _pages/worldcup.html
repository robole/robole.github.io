Motivation

The official Fifa website has a poor mobile user experience. It can be improved upon generally. I didn't see any other single website that offered a good guide to the world cup, along with live scores, and results. I wanted to see if I could make a happy home for all of that.

There are not many good APIs for world cup data. This was addressed partially through this REST API by the people at Software for Good. It scrapes data from the Fifa website and has been made availabe for free. It is well designed, but the performance is a bit sluggish. The data available is predominantly for matches, I will re-use some of it, and blend it with my own API.ğŸ¹

Finished Application

You can try it!

It is hosted on heroku using a free account. The website sleeps after 30 minutes of inactivity. So, it may take 2 minutes for the website to appear for you! Of course, when you pay for full account services, everything will work at light speed.
